# Веб-приложение «Дневник эмоций»

Разработка веб-приложения для комплексной оценки эмоционального состояния «Дневник эмоций».
Дипломная работа для программы Java-разработчик.

## Стек-технологий

- Java 17
- Maven
- Spring 3 (Boot, MVC, Data, Security)
- Hibernate
- Mysql
- Html, Bootstrap, CSS
